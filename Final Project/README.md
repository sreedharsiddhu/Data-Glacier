# Healthcare Persistency Analysis - Final Project

**Team:** The Closer  
**Member:** Rongala Sreedhar

## Project Overview
This project aims to analyze the `Healthcare Dataset` to identify the drivers of **Drug Persistency** (Persistent vs Non-Persistent) and build a machine learning model to predict it.

## Key Deliverables

### 1. Presentation
- [Final Presentation (PDF)](Final_Presentation.pdf)
- A slide deck summarizing our problem statement, methodology, EDA findings, and model recommendations.

### 2. Final Report
- [Final Project Report (PDF)](Final_Report.pdf)
- A detailed report covering the entire project lifecycle.

### 3. Code & Resources
- **EDA Plots**: [plots/](plots/)
- **Data**: `Healthcare_dataset.xlsx`
- **Scripts**: 
    - `generate_final_presentation.py`: Generates the slide deck.
    - `generate_final_report.py`: Generates the detailed report.
    - `create_plots.py`: Generates EDA visualizations.

## Weekly Breakdown
- **Week 10**: Data Setup & Initial EDA.
- **Week 11**: Deep Dive EDA & Visualization.
- **Week 12**: Model Training (Logistic Regression, RF, XGBoost) & Dashboard.

## How to Run
1. Install dependencies: `pip install pandas matplotlib seaborn fpdf`
2. Run plotting script: `python create_plots.py`
3. Generate deliverables: 
   - `python generate_final_presentation.py`
   - `python generate_final_report.py`
